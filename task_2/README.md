# Personal Profile Page

## Overview
This is a professional and modern personal profile webpage designed for developers and tech enthusiasts. It showcases your personal information, skills, and contact details in a clean, masculine, and tech-oriented UI.

## Features
- Responsive design for desktop and mobile.
- Modern grid and flex layouts for profile and skills sections.
- Animated entrance effects for text and images.
- Interactive skill cards with hover effects.
- Stylish contact form with hover effects on the button.
- Dark, tech-inspired color palette suitable for male professionals.

## Technology Stack
- HTML5
- CSS3 (including Flexbox, Grid, Animations)
- Vanilla JavaScript (for optional smooth scroll and interactions)

## Sections
1. **Header**: Displays your name prominently.
2. **Home/Profile Section**: Profile image with a short bio.
3. **About Me**: A detailed description of your professional focus.
4. **Skills**: Grid of core skills with hover animation.
5. **Contact**: Contact form with name, email, and message fields.
6. **Footer**: Contains copyright information.

## How to Use
1. Clone or download the repository.
2. Ensure `index.html`, `style.css`, and `script.js` are in the same folder.
3. Place your profile image inside an `assets` folder and update the path in `index.html`.
4. Open `index.html` in any modern web browser.

## Notes
- All animations and hover effects are done with CSS3.
- JavaScript is optional but included for smooth scroll enhancement.
- Fully responsive: stacks content vertically on smaller screens.
