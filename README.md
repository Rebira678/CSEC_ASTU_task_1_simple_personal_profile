
# simple Personal Profile Website

This is a modern personal profile website built using **only HTML, CSS, and JavaScript**.
It features a dark, masculine design with a side-image layout and an interactive contact form.

---

## Features

* Dark and attractive background design
* Side-by-side layout (text + image)
* Responsive for desktop and mobile
* Smooth scroll to contact section
* Contact form handled with JavaScript (actionable, no backend)

---

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla)

---

## Folder Structure

```
/project
│── index.html
|__ style.css
|__ script.js
│── assets/
│   └── my_photo.jpg
```

---

## How to Run

1. Download or clone the project
2. Place your image inside the `assets` folder
3. Open `index.html` in any modern browser

No installation required.

---

## Customization

* Change text content inside `index.html`
* Replace `assets/my_photo.jpg` with your own image
* Adjust colors by editing CSS variables in `:root`
